DebugFS Support¶

The QEP Linux Kernel Driver provides rich debugfs support. Along with configuration, status, and statistics, debug related information is also provided through the read-only interface. Debug information is provided for CMAC, QDMA, QEP and STM-N. The debugfs files are located at /sys/kernel/debug/qep_dev/ in the host filesystem. For every QEP ethernet device detected, a filesystem tree is populated with directory name as its <Bus:Device:Function>, herein after referred to as <BDF> in short.

# sudo tree /sys/kernel/debug/qep_dev/
