System Requirements
===================

Supported Xilinx Accelerator Card
---------------------------------

1. U250


Host System Configuration
-------------------------

QEP Linux kenel network driver release is verified on the following host system configuration.

+--------------------------+-------------------------------------------------------------+
|                          | Configuration Details                                       |
+==========================+=============================================================+
| Host System              | Supermicro SYS-5049A-TR                                     |
+--------------------------+-------------------------------------------------------------+
| Processor                | Intel Xeon Silver 4116 Processor                            |
+--------------------------+-------------------------------------------------------------+
| CPU Cores                | 24                                                          |                                       
+--------------------------+-------------------------------------------------------------+
| RAM                      | 64GB on local NUMA node                                     |
+--------------------------+-------------------------------------------------------------+
| PCIe                     | Gen 3 x16, Full height, Full Length, dual width slot        |
+--------------------------+-------------------------------------------------------------+
| Operating System         | Ubuntu 18.04.1 LTS                                          |
+--------------------------+-------------------------------------------------------------+
| Linux Kernel             | 4.15.0-65-generic                                           |
+--------------------------+-------------------------------------------------------------+
| Qemu Version             | QEMU emulator version 2.11.1(Debian 1:2.11+dfsg-1ubuntu7.3) |
+--------------------------+-------------------------------------------------------------+
| Vivado Version           | 2019.1                                                      |
+--------------------------+-------------------------------------------------------------+

QEP Linux kernel driver release is verified on the following guest system configuration
by attaching network PF to the guest VM

========================= ==================================
Guest System(VM)          Configuration Details
========================= ==================================
Operating System          Ubuntu 18.04 LTS
Linux Kernel              4.15.1-20-generic
RAM 					  24GB
Cores              		  16
========================= ==================================

A 100G NIC card from a different vendor is also required for checking connectivity with a peer NIC.
The release is verified using Mellanox ConnectX-5 as the peer NIC.

*Note*: The driver is not verified on other Linux distros and kernels
