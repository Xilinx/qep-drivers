######################################
Xilinx QEP Linux Kernel Network Driver
######################################

The Xilinx QDMA Ethernet Platform (QEP) adds Ethernet support to SDAccel. 

Xilinx provides a Linux kernel network driver that runs on a
PCI Express root port host PC to interact with QEP via PCI Express.
